# typingTestRecord
A standard for saving and replaying typing tests

## Event types
| type | meaning |
|------|---------|
| 0 | Test start |
| 1 | Correct letter |
| 2 | Incorrect letter | 
| 3 | Submit Correct Word |
| 4 | Submit Incorrect Word |
| 5 | Go back a letter |
| 6 | Go back a word |
| 7 | Delete an entire word | 
| 8 | Test end |
